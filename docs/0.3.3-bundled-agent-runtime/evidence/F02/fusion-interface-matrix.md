## Classification rules

- `DIRECT`: shape and runtime semantics are equivalent.
- `LOSSLESS_ADAPTER`: a deterministic adapter can preserve identity, order, correlation, and failure meaning.
- `SEMANTIC_REWRITE`: the target contract must enforce a different invariant or state machine.
- `UNSUPPORTED`: the current baseline cannot represent the source behavior safely.
- `UNKNOWN_CRITICAL`: required evidence or source closure is missing; F04 must not proceed.
