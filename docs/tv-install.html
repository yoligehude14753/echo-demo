EchoDesk 智能电视安装

适配 Android TV、Google TV 和常见国产 Android/AOSP 智能电视。电视上点 APK 会进入系统安装器；电脑端可以下载一键安装包，通过 ADB 推送到电视。

电视浏览器安装

用电视浏览器打开本页，遥控器选中下面按钮并按确认。下载后按系统提示安装， 安装完成后到电视的「我的应用」打开 EchoDesk。

下载电视 APK

电脑一键安装

下载 zip 后解压，打开电视的网络调试/ADB 调试，在电脑上运行安装脚本并输入电视 IP。

下载一键安装包
  1. 电视和电脑连接同一个网络。
  2. 电视开启开发者模式与 ADB 调试。
  3. macOS 运行 ./install-tv-macos.sh 电视IP。
  4. Windows 运行 install-tv-windows.ps1 -TvIp 电视IP。

复制 ADB 安装命令

已经下载并解压一键安装包后，可以在同目录运行下面的命令。电视弹出授权时先点允许。

一键安装默认是干净安装，避免会议室电视继承旧缓存。升级保留数据时，在命令前加 ECHODESK_TV_KEEP_DATA=1。
